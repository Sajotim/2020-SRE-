#### 题目：李玉堂学长的“简单”虚拟机

##### 测试环境：Visual Studio2013，win10

##### 语言：C ++

##### 实现功能：

| 指令  | 功能           |
| ----- | -------------- |
| Empty | 判断栈是否为空 |
| Size  | 栈的大小       |
| Plus  | 加             |
| Min   | 减             |
| Mul   | 乘             |
| Div   | 除             |
| Mod   | 取余           |
