# 红岩暑假作业

## 简单虚拟机 NAiVE-VM

**运行环境**：Python 3.8，WSL Ubuntu

**语言**：Python

**实现功能**：Level 1 实现一个简易的虚拟机

**介绍**：基于Python实现的栈式虚拟机(人菜，寄存器虚拟机确实做不来)，支持四则运算，需使用 [p-code 语言](https://en.wikipedia.org/wiki/P-code_machine) 输入

**example**

(541 * 114 - 4417) * 2 = 114514
